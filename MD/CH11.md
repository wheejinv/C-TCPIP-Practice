## 프로세스간 통신의 기본 개념.

두 프로세스간 통신을 위해서는 파이프라는 것을 생성해야 한다.<br>
이 파이프는 프로세스에 속하는 자원이 아니다.<br>
운영체제가 마련해 주는 메모리 공간을 통해서 두 프로세스는 통신을 하게 된다.

```c
#include <unistd.h>

// 성공 시 0, 실패 시 -1 반환
int pipe(int filedes[2]);
```

-   `filedes[0]` : 파이프의 출구. read 함수
-   `filedes[1]` : 파이프의 입구. write 함수
